
# SQL Analysis

This folder contains the SQL implementation for analyzing the Lending Club loan dataset.

## Folder Structure

### analysis/
Contains all the SQL queries used to answer the business questions.

- **question1_high_risk_segments.sql** – Identifies borrower and loan characteristics associated with higher bad loan percentages.
- **question2_portfolio_trend_analysis.sql** – Analyzes how the accepted loan portfolio evolved over time.
- **question3_geographic_distribution_analysis.sql** – Compares the geographic distribution of accepted and rejected loan applications.

Additional files:

- **findings.md** – Summary of key findings, business recommendations, and analysis limitations.
- **screenshots/** – Selected query outputs supporting the analyses.

### schema/
Contains SQL scripts used to build the clean and analytics layers from the raw Lending Club datasets.

---

The analyses were performed on the **analytics** schema after data cleaning and feature engineering.
